STYLING WEBSITES

CSS Flexbox and Grid layout help create responsive webpages.

Flexbox helps create a responsive layout through concentarting on making the content responsive while grid layout focuses on the layout rather than the content.

overrall both have their strengths and implementing both help create aresponsive website.

Flexbox Model

Flexbox helps create content that is responsive. it uses properties like :

  • justify-content.
  • flex-direction.
  • flex-wrap.
  • align-items.

These properties arrange the content along the main and cross axis of the flexbox.

To apply the flexbox to an element you apply the display css style to flex.

GRID LAYOUT

This model focuses on the layout unlike the flexbox model. It uses certain properties to style the layout, some properties include:

  • grid-template-column.
  • grid-template-row
  • gap

These are only some but they do create magic inside your webpages.

The grid column and grid row templates creates thr column and rows of the element. We assign the grid property to an element usind display property in css and setting it to grid.
